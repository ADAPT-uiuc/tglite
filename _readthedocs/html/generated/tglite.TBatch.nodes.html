tglite.TBatch.nodes¶

TBatch.nodes(include_negs=True, reverse=False) → ndarray¶

Returns a node index array: [src, des(, neg)] if reverse is False or [des, src(, src)] if reverse is True.

Parameters:

  • include_negs (bool) – Whether to include negative nodes.

  • reverse (bool) – Whether to reverse the edges.

Return type:

np.ndarray
