tglite.iter_edges¶

tglite.iter_edges(g: TGraph, size=1, start=None, end=None) → EdgesIter¶

Create and return an iterator to generate TBatch

Parameters:

  • g (TGraph) – The graph to iterate on.

  • size (int) – Number of edges in each mini-batch.

  • start (int or None) – The starting edge index.

  • end (int or None) – The ending edge index.

Return type:

EdgesIter
